## Задача

**Рівняння:** `y' = -2y + x`  
**Початкова умова:** `y(0) = 1`  
**Відрізок:** `[0, 1]`  
**Крок:** `h = 0.1`  
**Задана точність:** `ε = 1e-4`  

**Аналітичний розв'язок:**  
`y(x) = x/2 - 1/4 + (5/4)·exp(-2x)`

> Щоб змінити рівняння — відредагуйте `exact_solution.py` (функції `f()`, `y_exact()` та параметри `X0, Y0, X_END, H, EPS`).

## Використані формули

### Метод Рунге-Кутта 4-го порядку

```
k1 = f(x_n,         y_n)
k2 = f(x_n + h/2,   y_n + h/2·k1)
k3 = f(x_n + h/2,   y_n + h/2·k2)
k4 = f(x_n + h,     y_n + h·k3)
y_{n+1} = y_n + h/6·(k1 + 2·k2 + 2·k3 + k4)
```

### Метод прогнозу Адамса 2-го порядку

```
y*_{n+1} = y_n + h/2·(3·f_n - f_{n-1})
```

### Метод корекції Адамса 2-го порядку

```
y^{(k+1)}_{n+1} = y_n + h/2·(f(x_{n+1}, y^{(k)}_{n+1}) + f_n)
```

### Оцінка похибки (метод Рунге)

```
ε_n = |y_{n+1}(h) - y_{n+1}(h/2)| / (2^s - 1)
```
де `s` — порядок методу (2 для Адамса, 4 для РК-4).

### Правило вибору кроку

```
якщо ε > EPS           →  h = h / 2    (зменшити)
якщо ε ≤ EPS / 2^s     →  h = h · 2    (збільшити)
інакше                 →  h не змінювати
```
